# e2ev-framework

A Distributed Trust Framework for Verifiable Elections: Strong Software Independence via Hybrid Consensus.

This repository contains the formal specifications, threat models, and architectural definitions for an End-to-End Verifiable (E2E-V) voting framework. The system combines:
*   Cryptographic Authenticity (Blind Signatures & Merkle Logs)
*   Physical Sovereignty (Voter-Verified Paper Audit Trails - VVPAT)
*   Statistical Assurance (Risk-Limiting Audits - RLA)
*   Coercion Resistance (Deniable Verification & Receipt-Freeness)

## Documentation
All formal specifications and academic drafts are located in the `/docs` directory.

## Proof of Concept
A Java-based Invariant Tester is available in the `/poc-voting` directory, mathematically proving the core properties of the protocol.
